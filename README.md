# PHP School Management Website

A fully dynamic **school website built with PHP**, allowing administrators to manage content, students, staff, and more. Almost every part of the website can be changed via the admin panel. The system also supports **Google login** for quick authentication.

---

# Features

## Dynamic CRUD Operations

Administrators can perform **CRUD operations** for:

- Student records
- Teacher profiles
- Classes and courses
- Announcements and notices
- Events
- Frontend content (text, images, banners)

All changes are immediately reflected on the website.

---

## Frontend Customization

- Most sections of the website can be updated directly from the admin panel.
- Easily update:
  - Homepage content
  - Section titles and descriptions
  - Images, banners, and logos
- No coding required to make content changes

---

## Authentication

- Admin login for full access
- **Google login** integration
- Passwords stored securely with PHP hashing functions

---

## Responsive Design

- Fully responsive layout
- Works on desktops, tablets, and mobile devices
- Frontend built with HTML, CSS, and JS frameworks for modern UI

---

# How It Works

## 1. User Authentication

- Admin logs in with email/password or Google OAuth
- Secure password hashing prevents leaks
- Google login simplifies access and authentication

---

## 2. CRUD Management

- Admin panel provides interface for all content sections
- Add, edit, or delete entries easily
- Backend PHP scripts handle database operations

---

## 3. Frontend Rendering

- Data is fetched from the database using PHP
- Dynamic rendering ensures all updates are reflected immediately
- Supports image uploads and media content
